# Hauser – Website

Landing page for Hauser – Your real estate portfolio. Clear at a glance.

## Structure

- `index.html` – Main landing page
- `datenschutz.html` – Privacy policy (customize with your details)
- `impressum.html` – Legal notice (customize with your details)
- `vercel.json` – Optional config for Vercel deployment

## Deployment

### GitHub Pages

1. Push this folder to a new GitHub repository
2. Go to **Settings → Pages**
3. Under **Source**, select **Deploy from a branch**
4. Choose `main` branch and `/ (root)` folder
5. Save – your site will be live at `https://<username>.github.io/<repo-name>/`

### Vercel

1. Push to GitHub
2. Go to [vercel.com](https://vercel.com) and import your repo
3. Deploy – Vercel auto-detects static files

## Customization

Before going live, update:

- **Impressum** (`impressum.html`): Add your company name, address, USt-IdNr.
- **Datenschutz** (`datenschutz.html`): Add responsible party details
- **Legal contact**: info@hauser.app (Papa Company UG)
